# Smart Energy Saving

Welcome to Nokia Summer Practice 2026. This is the project starting point, from which you will start your work.

## Tech Stack

### Backend

- Python 3
- Flask
- Mongo DB

### Frontend

- React - web framework
- Vite - bundler and build system
- Material UI - UI/UX framework

## Get Started

### Prerequisites

- Install Python 3.10 or newer
- Install Node.js v22 or newer
- Install Mongo DB local server

### Backend

Install dependencies:

```sh
$ cd backend
$ python -m venv venv
$ source venv/bin/activate # Windows: venv/Scripts/activate
$ pip install -r requirements.txt
```

Start the backend:

```sh
$ flask run --debug

Default development user 'admin' created.
 * Debug mode: on
WARNING: This is a development server. Do not use it in a production deployment. Use a production WSGI server instead.
 * Running on http://127.0.0.1:5000
```

### Frontend

Install dependencies:

```sh
$ npm install
```

Start the frontend:

```sh
$ npm run dev
```
